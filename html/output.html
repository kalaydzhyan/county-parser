Description of the data fields

prop_id - property ID at the county database.
legal_description - legal description of the property. This field is extracted from the county database and is not necessarily accurate.
prop_address - property street address, if it has one.
owner_name - name of the owner.
owner_address - address of the owner.
absentee - whether or not the owner lives on the property (True=absentee, False=homestead).
empty_land - whether or not the land is empty/vacant. If total improvements are below $4000, the land still counts as empty.
property_use - property use, if any. Sometimes, same as zoning.
zoning - property zoning. See land_dict for Mixed zoning.
land_area - area of the land, in acres.
land_dict - dictionary of uses with corresponding land areas, when available.
recent_penalty - most recent tax penalty.
recent_delinq - most recent tax delinquency.
school - school district.
inactive - flag showing the property is inactive, if such information is supplied by the county.
